ストレスチェック集計自動化

カテゴリ: AI業務自動化 ｜ 使用環境: Claude Code / formrun / 厚労省版プログラム

課題（Before）

ストレスチェックの回答収集が紙ベースで、回収・転記・集計に大きな工数がかかっていた。厚生労働省版プログラムへのデータ取込も手作業だった。

やったこと

  • formrun の Web フォームで回答を収集する仕組みに切り替え
  • 回答データを厚労省版プログラムの取込形式 CSV へ自動変換
  • 配布 → 回収 → 転記の紙フローを丸ごとデジタル化

成果（After）

配布・回収・転記の工数を大幅に削減。転記ミスがなくなり、集計までの期間も短縮された。
